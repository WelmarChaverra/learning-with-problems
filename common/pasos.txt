Pasos del proceso con Git

1. Crear cuenta en GitHub

2. descargar Git

3. Crear una carpeta para trabajar el repositorio

4. Ingresar a la consola/Git Bush dando clic derecho en la carpeta creada.

5. Clonar el repositorio de Wel con la url tomada desde el mismo

$ git clone https://github.com/WelmarChaverra/learning-with-problems.git

6. ingresamos al repositorio clonado

$ cd learning-with-problems

7. Creamos un branch nuevo

$ git checkout -b feature/Michael

8. Creamos un archivo txt vacio llamado pasos en la carpeta common

9. $ git add common/pasos.txt

10. $ git commit -m "Archivo prueba inicial commit"

11. $ git push origin feature/Michael

12. luego, tras realizar cambios sobre el archivo pasos y guardarlos, repetimos pasos 9, 10 y 11
